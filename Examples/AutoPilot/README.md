**Auto Pilot**
This example section is for the NMEA 2000 autopilot.  The autopilot consists of a 433mhz receiver that can be used to control the NMEA 2000 autopilot.  Note the NMEA 2000 has only been tested with a RayMarine EV1 autopilot
